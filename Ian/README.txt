Setup guide:
Dubble click 'DynamDemoBal.exe' and it starts

Config guide:
If you want to config the values, you have to open the config with visual studio code and edit it there.

Recommended settings:
velocityXMin = 0.3		Min velocity of X axis
velocityXMax = 1.3		Max velocity of X axis
velocityYMin = 0.75		Min velocity of Y axis
velocityYMax = 2.5		Max velocity of Y axis
radiusMin = 75		Min Radius of ball
radiusMax = 95		Max Radius of ball
start_amount = 3		Amount of balls at the start

fullscreen = false		False for window mode - true for fullscreen
target_fps = 60		Changes target FPS
